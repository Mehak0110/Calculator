# Calculator App 🧮

A fully functional calculator web application created using HTML, CSS, and JavaScript.

## Features:
- Basic arithmetic operations (+, -, ×, ÷)
- Scientific calculator functions:
  - sin, cos, tan
  - square root
  - power (x²)
  - Pi (π)
- Percentage calculation
- Delete last input
- Clear screen
- Keyboard support
- Calculation history
- Responsive and modern UI design

## Technologies Used:
- HTML5
- CSS3
- JavaScript

## Purpose:
This project was built to practice JavaScript concepts like DOM manipulation, functions, event handling, and building interactive web applications.
